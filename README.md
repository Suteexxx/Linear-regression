# 📈 Linear Regression Project

## 📌 Overview
This project implements **Linear Regression**, a fundamental machine learning algorithm used for predicting continuous values. It models the relationship between input features and a target variable using a linear equation.

---

## 🧠 What is Linear Regression?

Linear Regression finds the best-fit line:

y = mx + c

- **y** → Dependent variable (output)
- **x** → Independent variable (input)
- **m** → Slope (coefficient)
- **c** → Intercept

---

## 🚀 Features

- Simple Linear Regression implementation
- Data visualization using matplotlib
- Model training using scikit-learn
- Performance evaluation (MSE, RMSE, R² Score)
